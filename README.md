# 2023 CPL

本目录收录了 2023 年 CPL 课程的作业、练习与示例代码。内容按章节组织，每个子目录对应一节课程。

主要内容

- 各章练习：类型与 I/O、循环、函数、递归、指针、链表等。
- 每个子目录包含若干示例代码和练习题的实现。
